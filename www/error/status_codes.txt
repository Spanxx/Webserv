## Successful
200 = OK
201 = Created
204 = No Content

## Redirection
301 = Moved Permanently
302 = Found
303 = See Other
304 = Not Modified
307 = Temporary Redirect

## Client Error
400 = Bad Request
403 = Forbidden
404 = Not Found
405 = Method Not Allowed
408 = Request Timeout
411 = Length Required
413 = Request Too Large
414 = URI Too Long
415 = Unsupported Media Type

## Server Error
500 = Internal Server Error
501 = Not Implemented
502 = Bad Gateway
503 = Service Unavailable
504 = Gateway Timeout
